Уровень 2 (CSS2)
Блочная вёрстка. 
Появились относительное, абсолютное и фиксированное позиционирование. 
Позволяет управлять размещением элементов по странице без табличной вёрстки.

Наследование. Каскадирование. Приоритеты стилей CSS

Виды селекторов:

Универсальным селектор
* {
   margin: 0; 
   padding: 0; 
}

Селектор элементов
p
{
   font-family: arial, helvetica, sans-serif; 
}

Селектор классов
.note 
{
   /.Перед точкой можно указывать тег, тогда класс будет распространяться только на этот элемент. ./
   color: red; 
   background-color: yellow; 
   font-weight: bold; 
}

Селектор идентификаторов
#paragraph1 
{
   /.Перед точкой можно указывать тег, тогда класс будет распостроняся только на этот элемент. ./
   margin: 0px; 
}

Селекторами атрибутов
a[href="http://www.somesite.com"] 
{
   font-weight: bold; 
}

Селектор потомков (контекстный селектор)
div#paragraph1 p.note
 {
   color: red; 
}

Селектор дочерних элементов
p.note > b 
{
color: green; 
}

Селектор сестринских элементов
h1 + p 
{
   font-size: 24pt; 
}

Селектор псевдоклассов
a:active 
{
   color: yellow; 
}

Селектор псевдоэлементов
p::first-letter 
{
   font-size: 32px; 
}
